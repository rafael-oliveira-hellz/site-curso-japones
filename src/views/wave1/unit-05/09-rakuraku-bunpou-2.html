らくらく文法2

Gramática Descomplicada #02

じょし　「の」

Partícula "no"

日本語で

ポルトガル語　では、何か　が　誰か　の　ものである　と　いいたい　とき　は、しょゆうかく　の　だいめいし　（meu, minha, seu, sua, tua, teu　など）　を　しよう　できます。　しかし、日本語　では　そう　では　ありません。じょし　「の」　は、それ　が　ぞくする　たんご　の　ちょくご　に　しよう　します。　これ　は、　ポルトガル語　の　「de」　と　すこし　にて　います。

例:　先生　の　本。(しかし、ちゅうい　して　ください、　それ　は　ポルトガル語　で　の　はんたい　です。)

Em Português:

Na língua portuguesa, quando queremos dizer que algo pertence a alguém, podemos utilizar os pronomes possessivos (ex.: meu, minha, seu, sua, teu, tua). Mas, na língua japonesa, não é assim. Usamos a partícula の logo depois da palavra a quem pertence. Funciona um pouco parecido com o "de" do português.
Ex: O livro do professor. (Mas cuidado, é o inverso do português.)


    例を参照してください　(Veja os exemplos):

  1. ダニロさん の ゲーム です。
    É o jogo do Danilo.
  2. 私 の 家 家　=　casa です。
    É a minha casa.
  3. 私 の ノート ノート　=　caderno です。
    É o meu caderno.

  4. 先生 の きょうかしょ きょうかしょ　=　livro didático です。
    É o livro didático do professor.


  5. フェリッペさん の パーティー です。
    É a festa do Felipe.


  6. これ　は 私 の です。
    Isso é meu.
  7. これ　は ダニエラさん　の じしょ じしょ　=　dicionário です。
    Esse é o dicionário da Daniela.
  8. この スマホ　は スマホ　=　smartphone リンさん　の です。
    Esse smartphone é da Lin.
  9. あした、 あね　の パソコン　を つかいます。
    Amanhã, vou usar o computador da minha irmã mais velha.
  10. 母　は 母　=　mamãe, minha mãe 弟　の 弟　=　irmão mais novo たんじょうび　ケーキ　を たんじょうび　ケーキ　=　bolo de aniversário つくります。 つくります　=　fazer, preparar
    A minha mãe fará o bolo de aniversário do meu irmão mais novo.
  11. あに の 車　は フェラーリ です。
    O carro do meu irmão mais velho é uma Ferrari.
  12. まいにち、 まいにち　=　todos os dias 父　の しんぶん　を しんぶん　=　jornal よみます。
    Todo os dias, leio o jornal do meu pai.
