日本語会話2

Diálogo em Japonês #02

だれ　の　本　です　か

De quem é esse livro?

Alguma vez você já encontrou algum objeto perdido? Um guarda-chuva, um livro, uma carteira? Nesta aula, o Felipe encontra um livro e conversa com sua amiga sobre quem poderia ser o dono! Aproveite para revisar os conteúdos sobre perguntas e ampliar seu vocabulário.


会話

Walk 'n' Talk

  1. あれ? だれ　か ここ　に 本　を わすれました。 わすれました　=　(passado) esquecer
    Ué? Alguém esqueceu um livro aqui.


  2. え？ フェリッペさん　の じゃない　です か。
    Ué? Não é seu, Felipe ?


  3. 私　の？ 私　の じゃない　です。
    Meu? Não é meu.


  4. そうですか。 そうですか　=　É mesmo? 何　の 本 です か。
    É mesmo? Do que é o livro ?


  5. えっと... かんこくご　の かんこくご　=　coreano 本 みたい みたい　=　parece です。
    Deixe eu ver... Parece ser um livro de coreano.


  6. かんこくご　の です か。 ミンさん　の 本 じゃない　です か。
    De coreano ? Não é o livro da Ming ?


  7. え？ でも、 ミンさん　は けっせき　です けっせき　=　ausente よね。
    Ué? Mas, a Ming faltou, né?


  8. じゃあ、 いちおう しまって しまって　=　forma "-te" do verbo しまう　- deixar おきます。 おきます　=　guardar あと　で、 みんな　に ききます。 ききます　=　perguntar
    É mesmo... Então, vou deixar guardado. Depois, pergunto ao pessoal.
