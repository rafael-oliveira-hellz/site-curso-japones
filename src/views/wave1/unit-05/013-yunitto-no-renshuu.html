ユニットの練習

Exercícios da Unidade

日本語　会話　1　ー　かんげいかい

Diálogo em Japonês #01 - Festa de boas-vindas

  1. Qual das alternativas abaixo resume adequadamente a ideia central trabalhada no primeiro diálogo?
    1. Convite de um(a) conhecido(a) para ir a uma festa
    2. Convite de um(a) melhor amigo(a) para ir a uma festa
    3. Um funcionário convida seu chefe para ir a uma festa
    4. Um aluno convida o seu professor para ir a uma festa

  2. Neste diálogo, Nanami fala sobre a festa de boas-vindas com Felipe e ele pergunta quando será essa festa. O que ele fala em japonês?
    1. 行きます　か。
    2. 何　です　か。
    3. どこ　です　か。
    4. いつ　です　か。

  3. No diálogo, Nanami fala sobre a festa de boas-vindas. Felipe não entende e pergunta o que é. Como ele diz isso?
    1. かんげいかい　は　どこ　です　か。
    2. かんげいかい　は　いつ　です　か。
    3. かんげいかい　って　何　です　か。
    4. かんげいかい　へ　行きます　か。

らくらく文法1　ー　「か」のじょしとぎもんし

Gramática Descomplicada #01 - Partícula "か" e pronomes interrogativos

  1. Veja a frase abaixo:
    ⇒　エリナさん　は　スペイン語　を　べんきょう　します。
    ⇒　A Erina estuda espanhol.
    Como podemos transformar essa frase em uma pergunta?
    1. エリナさん　は　スペイン語　を　べんきょう　します　よ。
    2. エリナさん　は　スペイン語　を　べんきょう　します　か。
    3. エリナさん　は　スペイン語　を　べんきょう　します　が。
    4. エリナさん　は　スペイン語　か　べんきょう　します。

  2. Qual das alternativas abaixo pergunta sobre um lugar?
    1. この お店 は どこですか。
    2. 駅 へ 行きます か。
    3. かいしゃ って 何 です か。
    4. 学校 で しごと します か。

  3. Imagine a seguinte situação: em uma festa, seu amigo pediu que você procure uma pessoa que você não conhece, o Sawada. O que você poderia dizer?
    1. さわださん は 何 です か。
    2. さわださん は 行きます か。
    3. さわださん は 何人 です か。
    4. さわださん は 誰 です か。

発音タイム　ー　そくおん

Hora da Pronúncia - Consoantes duplas

  1. Qual seria a transcrição correta da frase abaixo em roomaji:
    子ども　の　とき、この　学校　に　かよった
    1. Kodomo no toki, kono gakou ni kayota.
    2. Koddomo no toki, kono gakkou ni kayotta.
    3. Kodomo no toki, kono gakkou ni kayotta.
    4. Kodomo no toki, kono gatsukou ni kayotsuta.

  2. Veja a frase abaixo em roomaji e marque a opção correta em japonês:　
    Kono koppu wa ippai desu.
    1. この コプ は いぱい です。
    2. この コップ は いっぱい です。
    3. この コツプ は いつぱい です。
    4. この コープ は いいぱい です。

  3. 書き取りタイム! (Kakitori taimu! - Hora do ditado!) Ouça a frase (áudio 1) estudada em aula, e escreva em hiragana e katakana o que ouviu no espaço abaixo.

      ぶっか が ご パーセント さがりました。

日本語 会話 2　ー　誰　の　本　です　か

Diálogo em Japonês #02 - De quem é esse livro?

  1. Qual das alternativas abaixo resume adequadamente a ideia central trabalhada no segundo diálogo?
    1. Conversa entre conhecidos sobre o autor de um livro
    2. Conversa entre conhecidos sobre um objeto esquecido
    3. Conversa entre conhecidos sobre uma pessoa desaparecida
    4. Conversa entre professores sobre uma tarefa esquecida
  2. Ao ser questionado se o livro pertence ao Felipe, ele nega. Como ele fala isso?
    1. 私　の　です。
    2. 何　の　本　です　か。
    3. 私　の　じゃない　です。
    4. ミンさん　の　本　じゃない　です　か。
  3. Ao saber que o livro não é do Felipe, sua amiga pergunta do que é esse livro. Como ela pergunta isso?
    1. フェリッペさん　の　じゃない　です　か。
    2. 何　の　本　です　か。
    3. そう　です　か。
    4. ミンさん　の　本　じゃない　です　か。

らくらく文法　2　ー　じょし「の」

Gramática Descomplicada #02 - Partícula "no"

  1. Qual das frases abaixo apresenta o uso correto da partícula の?
    1. これ は 本 の 日本語 です。
    2. これ は 日本語 の 本 です。
    3. これ は 本 日本語 です の。
    4. これ の 日本語 は 本 です。

  2. Veja as palavras abaixo:
    です　フェリッペさん　パソコン　この
    Qual das opções abaixo está na ordem correta?
    1. この フェリッペさん の パソコン です。
    2. フェリッペさん の パソコン この です。
    3. この パソコン フェリッペさん です。
    4. この パソコン は フェリペさん の です。

  3. Imagine a seguinte situação: você encontra uma carteira no chão (em japonês, さいふ). A carteira está sem documento, logo, você não sabe de quem é. O que você poderia perguntar a seus colegas para descobrir a quem pertence?
    1. この さいふ は 誰 の です か。
    2. この 誰 は さいふ の です か。
    3. の 誰 は この さいふ です か。
    4. この さいふ は 何 の です か。

ネイティブ日本語　ー　わからないときのひょうげん

Japonês com Nativo - O que dizer quando não entende algo

  1. Quais das frases abaixo, indicam que você não entendeu o que foi falado em japonês?
    1. はい、そうです。
    2. はい、わかりました。
    3. すみません、わかりません でした。
    4. すみません、どうぞ。

  2. Veja o diálogo abaixo:

    A: この レポート を ださない と 行きません。
    B: _______________________________.
    A: この レポート を ださいと 行きません。

    Qual das opções abaixo pode completar corretamente a lacuna?
    1. はい、わかりました。
    2. そう です か。
    3. そう です ね。
    4. すみません、もう　いちど　おねがい　します。

  3. Veja a frase abaixo:
